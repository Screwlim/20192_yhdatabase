01 Node				This is the number of the sending node
				송신 노드 번호

02 Utilized Bandwidth Rate		This is the normalization of "Used Bandwidth"
				Used Bandwidth 정규화

03 Packet Drop Rate			This is the normalization of "Percentage of Lost Pcaket Rate"
				Percentage of Lost Packet Rate 정규화

04 Reserved Bandwidth		Initial reserved Bandwidth assigned (given) to each node,
				the user (usr) in the experiments assign these values
				처음에 할당 받은 Bandwidth

05 Average Delay Time per sec		Average Delay Time (per second) for each node
				This is (End-to End Delay)
				End to End 노드 당 평균 지연 시간

06 Percentage of Lost Packet Rate	Percentage of Packets Drop Rate for each node
				노드 당 패킷 손상 비율

07 Percentage of Lost Byte Rate	Percentage of Lost Byte Rate for each node
				노드 당 바이트 손상 비율

08 Packet Received Rate		Total received packets (per second) for each node based on "Reserved Bandwidth"
				할당 Bandwidth에서의 초당 수신 패킷 비율

09 Used Bandwidth			This is what each node could reserve from the "Reserved Bandwidth"
				할당 Bandwidth에서의 Bandwidth 사용량

10 Lost Bandwidth			The amount of lost Bandwidth by each node from "Reserved_Bandwidth"
				할당 Bandwidth에서의 Bandwidth 손상량

11 Packet Size Byte			Packets size in Byte assigned specifically for each node to transmit
				Note: 60 Byte will be added to the 1440 for the IP Header and the UDP Header
				((Data size 1440 Byte) + (IP Header 40 Byte) + (UDP Header 20 Byte)) =1500 Byte
				Packet 크기

12 Packet Transmitted		Total transmitted packets (per second) for each node based on the "Reserved Bandwidth"
				할당 Bandwidth서의 초당 패킷 전송량

13 Packet Received			Total received packets (per second) for each node based on the "Reserved Bandwidth"
				할당 Bandwidth에서의 초당 패킷 송신량

14 Packet Lost			Total lost packets (per second) for each node, which based on the "Lost Bandwidth"
				할당 Bandwidth에서의 초당 패킷 손상량

15 Transmitted Byte			Total transmitted Byte (per second) for each node
				노드 당 초당 바이트 전송량

16 Received Byte			Total received Byte (per second) for each node based on the "Reserved_Bandwidth"
				할당 영역에서의 노드 당 초당 바이트 송신량

17 10-Run AVG Drop Rate		Average packet drop rate for 10 consecutive (run) iterations
				평균 패킷 손상 비율 10회

18 10-Run AVG Bandwidth Use	Average Bandwidth utilized for 10 consecutive (run) iterations
				평균 Bandwidth 사용량 10회

19 10-Run Delay			Average delay time for 10 consecutive (run) iterations
				평균 지연 시간 10회

20 Node Status			initial classification of nodes based on
				"Packet Drop Rate", "Used Bandwidth" and "Average Delay Time per Sec"
				B = Behaving  |  NB = Not Behaving  |  P NB = Potentially Not Behaving
				패킷 손상, Bandwidth 사용량, 지연 시간 기반, 분석 결과 (B, NB, PB)

21 Flood Status			Percentage of flood per node based on "Packet Drop Rate"
				Medium and high level of BHP flood attack in case B
				B인 경우, 패킷 손상 기반, 노드 당 Flood 확률

22 Class				{NB-No Block, Block, No Block, NB-Wait} This is for case B
				The final classification of nodes based on
				"Packet Drop Rate", "Reserved Bandwidth", "Iteration", "Used_Bandwidth"
				B인 경우, 패킷 손상, 반복 작업, Bandwidth 사용량 기반, 최종 분류
